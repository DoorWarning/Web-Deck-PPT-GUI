# WebDeck — 인터랙티브 웹페이지형 발표 도구

PowerPoint 클론이 아니라, **웹페이지처럼 동작하는 인터랙티브 발표 자료**를 만드는 도구입니다.
보는 사람이 실시간으로 직접 조작(차트 슬라이더, 폴/퀴즈, 라이브 코드 실행)할 수 있고,
HTML/CSS/JS 기반이라 애니메이션·임베드·커스텀 코드·수식이 자유롭습니다. 백엔드 없이 단일
뷰어 로컬 인터랙션으로 동작하며, 정적 파일 하나로 배포(GitHub Pages 등)·임베드할 수 있습니다.
발표 런타임은 [reveal.js](https://revealjs.com/) 위에서 돌아갑니다.

## 모델: 섹션 + 블록

```
Deck → Section[] → Block[]
```

- **Deck** — `canvas{w,h}`(발표 해상도/비율) + `theme` + `sections`.
- **Section** = 발표의 한 장면(슬라이드). 좌우(→/←)로 전환.
- **Block** = 콘텐츠 단위. 기본은 **자유 배치(absolute, %좌표)** — 어디에나 두고 크기를 조절.
  섹션 레이아웃을 `가운데/흐름/2단/그리드`로 바꾸면 블록이 자동 정렬(반응형)됩니다.
- 좌표·크기가 % 기반이라 캔버스 비율을 바꿔도 함께 맞춰집니다.

## 블록 (인터랙션)

| 블록 | 설명 |
|------|------|
| 텍스트 | 마크다운 + **LaTeX 수식**(`$x^2$`, `$$…$$`, KaTeX) |
| 미디어 | 이미지/비디오 |
| 차트 | Chart.js — **슬라이더로 데이터 실시간 조작** |
| 폴/퀴즈 | 로컬 집계 막대·정답 피드백 |
| 플레이그라운드 | **뷰어가 코드 수정·실행** (sandbox iframe) |
| 임베드 | iframe |
| 커스텀 코드 | 원시 HTML/CSS/JS 주입 (Chart.js/Three.js CDN 등 자유) |

각 블록은 **등장 효과(reveal.js fragment 단계·스크롤 트리거)**와 **인터랙션**(클릭 → 섹션
이동·블록 토글·URL·변수·스크립트)을 가질 수 있습니다. 블록별 **폰트·색·텍스트 정렬**도 지정 가능.

## 편집 조작

- **이동**: 블록을 아무 데나 잡아 드래그. 가운데/가장자리 **스냅 + 가이드 선**, 인스펙터 `정렬` 버튼.
- **크기 조절(핸들)**: 코너(대각선)=내용까지 비율 유지 확대/축소, 좌우 변=폭(텍스트 줄바꿈),
  상하 변=높이. **중심 기준**으로 조절.
- **단축키**: `Ctrl+Z`/`Ctrl+Shift+Z` 실행취소·재실행, `Ctrl+D` 복제, `Delete` 삭제.

## 개발 / 빌드 / 배포

```bash
npm install
npm run dev      # 저작(에디터) — 섹션/블록 추가, 라이브 프리뷰
npm run build    # dist/index.html — 단일 자체완결 파일(외부 참조 0, 파비콘 인라인)
npm run preview  # 빌드 결과 미리보기
```

- `dist/index.html` 한 장을 그대로 호스팅하면 배포 완료(GitHub Pages·Netlify·블로그 임베드).
- 배포 링크에 `?present` 또는 `#present`를 붙이면 **발표 모드로 바로 시작**.
- 툴바의 **💾 HTML** 버튼은 현재 덱이 박힌 **편집 가능한 단일 HTML**을 내려받습니다
  (받은 사람이 다시 열어 편집 — 자기복제형). **JSON 저장/불러오기**도 지원.

## 저작 방식 (하이브리드)

- **GUI**: 왼쪽에서 섹션·블록 추가/재정렬, 가운데 라이브 프리뷰, 오른쪽 인스펙터로 설정.
- **코드 주입**: `플레이그라운드`/`커스텀 코드` 블록으로 임의 HTML/CSS/JS 삽입.
- **AI 저작**: 툴바의 **`?` 도움말**에 복사용 AI 프롬프트가 있습니다. [`AUTHORING.md`](./AUTHORING.md)
  와 `src/model/schema.ts`(JSON Schema)를 Claude 등에게 주면 덱 JSON을 생성·수정할 수 있고,
  결과를 **불러오기**로 적용합니다. (도움말 팝업에서 두 파일을 바로 내려받을 수 있어요.)

## 발표 모드 (reveal.js)

→/Space 다음(단계 포함) · ← 이전 · Esc 또는 O 오버뷰 · F 전체화면 · S 발표자 뷰 · Q 편집 복귀.
차트·폴/퀴즈·플레이그라운드는 발표 중 청중이 직접 조작할 수 있습니다.

## 구조

| 영역 | 위치 |
|------|------|
| 데이터 모델·직렬화·스키마 | `src/model/` |
| 상태·undo/redo | `src/store/store.ts` |
| 블록 라이브러리 + 레지스트리 | `src/blocks/` |
| 공유 렌더(섹션/블록·드래그/리사이즈) | `src/render/` |
| 저작 UI(툴바·패널·인스펙터·도움말) | `src/editor/` |
| 발표 런타임(reveal.js) | `src/present/Presenter.tsx` |
| 인터랙션·Deck API·sandbox | `src/runtime/` |
| 단일 HTML·JSON 입출력 | `src/io/` |

주요 의존성: React + Vite(`vite-plugin-singlefile`), reveal.js, Chart.js, KaTeX, marked + DOMPurify, zustand.

## 보안 메모

`플레이그라운드`/`커스텀 코드`/`runScript`는 코드를 실행합니다. 코드 블록은 `sandbox` iframe으로
격리되지만, 커스텀 스크립트(`scripts[]`)는 페이지 컨텍스트에서 실행되므로 **신뢰하는 코드만**
넣으세요. 외부에서 받은 덱을 발표할 땐 스크립트를 확인하세요.
