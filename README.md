# WebDeck — 인터랙티브 웹페이지형 발표 도구

PowerPoint 클론이 아니라, **웹페이지처럼 동작하는 인터랙티브 발표 자료**를 만드는 도구입니다.
보는 사람이 실시간으로 직접 조작(차트 슬라이더, 폴/퀴즈, 라이브 코드 실행)할 수 있고,
HTML/CSS/JS 기반이라 애니메이션·임베드·커스텀 코드가 자유롭습니다. 백엔드 없이 단일 뷰어
로컬 인터랙션으로 동작하며, 정적 파일로 배포(GitHub Pages 등)·임베드할 수 있습니다.

## 모델: 섹션 + 블록 (반응형)

PPT식 절대좌표가 아니라 **문서형 블록 모델**입니다.

```
Deck → Section[] → Block[]
```

- **Section** = 발표의 한 장면(좌우 → 로 전환, 넘치면 세로 스크롤).
- **Block** = 반응형 흐름/그리드로 배치되는 콘텐츠 단위.
- 레이아웃은 고정 1280×720 스케일이 아니라 **반응형**(모바일 대응).

## 블록(인터랙션)

| 블록 | 설명 |
|------|------|
| 텍스트 | 마크다운 |
| 미디어 | 이미지/비디오 |
| 차트 | Chart.js — **슬라이더로 데이터 실시간 조작** |
| 폴/퀴즈 | 로컬 집계 막대·정답 피드백 |
| 플레이그라운드 | **뷰어가 코드 수정·실행** (sandbox iframe) |
| 임베드 | iframe |
| 커스텀 코드 | 원시 HTML/CSS/JS 주입 (Chart.js/Three.js CDN 등 자유) |

각 블록은 **등장 효과(fragment 단계·스크롤 트리거)**와 **인터랙션**(클릭 → 섹션 이동·블록
토글·스크립트 실행 등)을 가질 수 있습니다.

## 개발 / 빌드 / 배포

```bash
npm install
npm run dev      # 저작(에디터) — 섹션/블록 추가, 라이브 프리뷰
npm run build    # dist/index.html — 단일 자체완결 파일(외부 참조 0)
npm run preview  # 빌드 결과 미리보기
```

- `dist/index.html` 한 장을 그대로 호스팅하면 배포 완료(GitHub Pages·Netlify·블로그 임베드).
- 배포 링크에 `?present` 또는 `#present`를 붙이면 **발표 모드로 바로 시작**.
- 툴바의 **💾 HTML** 버튼은 현재 덱이 박힌 **편집 가능한 단일 HTML**을 내려받습니다
  (받은 사람이 다시 열어 편집 — 자기복제형). **JSON 저장/불러오기**도 지원.

## 저작 방식 (하이브리드)

- **GUI**: 왼쪽에서 섹션·블록 추가/재정렬, 가운데 라이브 프리뷰, 오른쪽 인스펙터로 설정.
- **코드 주입**: `플레이그라운드`/`커스텀 코드` 블록으로 임의 HTML/CSS/JS 삽입.
- **AI 저작**: [`AUTHORING.md`](./AUTHORING.md) + `src/model/schema.ts`(JSON Schema)를
  Claude 등에게 주면 덱 JSON을 생성·수정할 수 있습니다. 결과는 "불러오기"로 적용.

## 구조

| 영역 | 위치 |
|------|------|

## 발표 단축키

→/Space/PageDown 다음(단계 포함) · ←/PageUp 이전 · Home/End 처음·끝 · F 전체화면 · Esc 종료

## 보안 메모

`플레이그라운드`/`커스텀 코드`/`runScript`는 코드를 실행합니다. 코드 블록은 `sandbox` iframe으로
격리되지만, 커스텀 스크립트(`scripts[]`)는 페이지 컨텍스트에서 실행되므로 **신뢰하는 코드만**
넣으세요. 외부에서 받은 덱을 발표할 땐 스크립트를 확인하세요.
